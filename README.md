# SDV_Dwarven_Expansion_Bundles
A collection of adjustments to the Community Center bundles to add or alter required items to include products from Dwarven Expansion.

Should only be used on a new save (as Bundles are set when the save is started) and is not compatable with other Bundle mods that alter the basic community center bundles.

This is released as an add-on mod so you can enable or disable it easily before a new game is started.
